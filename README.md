# Catalog Next

Простой пример каталога на Next.js + TypeScript.

Фичи:

- Загрузка данных с `https://fakestoreapi.com`
- Infinite scroll (подгрузка при скролле)
- Кэширование с `@tanstack/react-query`
- Компоненты документированы в Storybook
- Применение системной темы

Установка и запуск:

```bash
npm install
npm run dev
```

Storybook:

```bash
npm run storybook
```
